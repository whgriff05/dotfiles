# NVIM install guide

## Dependencies
- Neovim 0.11+
- Cargo
- curl
- git
- tree-sitter-cli (`cargo install --locked tree-sitter-cli`)
- node/npm  

## Important keybinds to remember
__leader = " "__

### Telescope
- "<leader>ff" find file
- "<leader>fg" live grep
- "<leader>fb" find buffer
- "<leader>fh" find help

### VimTex
- "\ll" start/stop auto-compilation

### Misc
- "<leader>cd" neovim file explorer
